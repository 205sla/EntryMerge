# ENT File Merge Tool

여러 개의 .ent 파일을 하나로 병합하는 도구입니다.

## 설치 방법

1. [Releases](https://github.com/yourusername/EntryMerge/releases) 페이지에서 최신 버전 다운로드
2. `ENT File Merge Tool Setup.exe` 실행
3. 설치 마법사 따라하기

## 사용 방법

1. 프로그램 실행
2. .ent 파일들을 드래그 앤 드롭하거나 파일 선택
3. '파일 결합' 버튼 클릭
4. 병합된 파일 저장
